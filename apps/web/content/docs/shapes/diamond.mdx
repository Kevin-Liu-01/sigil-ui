---
title: Diamond
description: A rotated-square diamond shape with configurable size, color, and optional animation. Perfect for decorative accents and status indicators.
---

## Import

```tsx
import { Diamond } from "@sigil-ui/components";
```

## Usage

```tsx
<Diamond />
```

Renders a 45-degree rotated square using CSS `clip-path`. Default size is `24px`.

## Sizes

```tsx
<Diamond size={12} />
<Diamond size={24} />
<Diamond size={48} />
<Diamond size={64} />
```

Or use a string for CSS units:

```tsx
<Diamond size="2rem" />
```

## Colors

```tsx
{/* Token-driven (default) */}
<Diamond color="var(--s-primary)" />

{/* Direct color */}
<Diamond color="oklch(0.65 0.17 160)" />

{/* Semantic token */}
<Diamond color="var(--s-success)" />
<Diamond color="var(--s-error)" />
<Diamond color="var(--s-warning)" />
```

## Filled vs outlined

```tsx
{/* Filled (default) */}
<Diamond variant="filled" color="var(--s-primary)" />

{/* Outlined */}
<Diamond variant="outlined" color="var(--s-primary)" />
<Diamond variant="outlined" strokeWidth={2} />
```

## Animation

### Pulse

```tsx
<Diamond animated="pulse" color="var(--s-success)" />
```

A gentle scale pulse using `--s-duration-slow` and `--s-ease-spring`.

### Spin

```tsx
<Diamond animated="spin" color="var(--s-primary)" />
```

Continuous rotation using `--s-duration-slower`.

### Float

```tsx
<Diamond animated="float" color="var(--s-secondary)" />
```

Subtle vertical float for decorative use.

## Status indicator pattern


## Decorative scatter pattern

```tsx
<div className="relative h-64">
  <Diamond size={48} color="var(--s-primary-muted)" className="absolute top-4 left-8" />
  <Diamond size={24} color="var(--s-primary)" className="absolute top-12 right-16" />
  <Diamond size={32} color="var(--s-secondary)" className="absolute bottom-8 left-1/2" />

```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-primary` | Default fill color |
| `--s-border` | Outlined variant stroke |
| `--s-duration-slow` | Pulse animation duration |
| `--s-duration-slower` | Spin animation duration |
| `--s-ease-spring` | Pulse easing |
| `--s-ease-default` | Float easing |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `number \| string` | `24` | Width and height |
| `color` | `string` | `var(--s-primary)` | Fill or stroke color |
| `variant` | `"filled" \| "outlined"` | `"filled"` | Visual variant |
| `strokeWidth` | `number` | `1.5` | Stroke width for outlined variant |
| `animated` | `"pulse" \| "spin" \| "float" \| false` | `false` | Animation type |
| `className` | `string` | — | Additional CSS classes |

Plus all standard `<div>` HTML attributes.

## Accessibility

- Renders with `role="presentation"` by default (decorative)
- Add `role="img"` and `aria-label` when the diamond conveys meaning (e.g. status indicator)
