---

## Import

```tsx
import { DiagramNode } from "@sigil-ui/components";
```

## Usage

```tsx
<DiagramNode
  label="API Server"
  sublabel="Node.js"
  variant="highlighted"
  size="md"
  badge={3}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | Primary node label |
| `sublabel` | `string` | — | Secondary label below |
| `icon` | `ReactNode` | — | Icon element |
| `variant` | `"default" \| "highlighted" \| "muted" \| "accent"` | `"default"` | Visual style |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Node size |
| `mono` | `boolean` | `false` | Use monospace font |
| `badge` | `ReactNode` | — | Badge shown at top-right corner |

## Token integration

| CSS Variable | Purpose |
