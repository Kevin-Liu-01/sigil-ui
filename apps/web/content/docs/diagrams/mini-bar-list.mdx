## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `items` | `MiniBarItem[]` | — | Ranked items |
| `valueLabel` | `string` | `"Count"` | Header for value column |
| `formatValue` | `(v: number) => string` | — | Custom value formatter |
| `color` | `string` | `var(--s-primary)` | Bar color |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default bar color |
| `--s-text` | Item label and value |
| `--s-text-muted` | Column headers and icons |
| `--s-font-mono` | Value font |
| `--s-radius-sm` | Bar radius |
