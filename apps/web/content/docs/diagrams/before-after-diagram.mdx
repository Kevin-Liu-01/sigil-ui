## Import

```tsx
import { BeforeAfterDiagram } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <BeforeAfterDiagram
    beforeTitle="Before"
    afterTitle="After"
    before="Manual CSS overrides, scattered Tailwind classes, no consistency"
    after="Token-driven design, one preset change updates everything"
  />
</ComponentPreview>

## Usage

```tsx
<BeforeAfterDiagram
  beforeTitle="Before"
  afterTitle="After"
  before="Manual edits"
  after="Automated tokens"
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `beforeTitle` | `string` | `"Without"` | Before panel label |
| `afterTitle` | `string` | `"With"` | After panel label |
| `before` | `ReactNode` | — | Before panel content |
| `after` | `ReactNode` | — | After panel content |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Panel borders |
| `--s-surface` | Panel background |
| `--s-error` | Before accent |
| `--s-success` | After accent |
| `--s-text` | Title and content |
| `--s-radius-md` | Panel radius |
