---
title: FlowDiagram
description: A directional flow chart with auto-layout, branching, and animated edge paths. Built on the Diagram primitive.
---

## Import

```tsx
import { FlowDiagram, FlowStep, FlowBranch } from "@sigil-ui/components";
```

## Usage

```tsx
<FlowDiagram direction="horizontal">
  <FlowStep id="start" label="User visits" />
  <FlowStep id="auth" label="Authenticate" />
  <FlowStep id="dashboard" label="Dashboard" />
</FlowDiagram>
```

Steps are automatically connected with directional arrows.

## Branching

```tsx
<FlowDiagram direction="vertical">
  <FlowStep id="input" label="Request" />
  <FlowBranch label="Valid?">
    <FlowStep id="success" label="Process" />
    <FlowStep id="error" label="Return error" />
  </FlowBranch>
  <FlowStep id="response" label="Send response" />
</FlowDiagram>
```

## Animated edges

```tsx
<FlowDiagram animated>
  <FlowStep id="a" label="Fetch data" />
  <FlowStep id="b" label="Transform" />
  <FlowStep id="c" label="Render" />
</FlowDiagram>
```

Dashed edges animate to show flow direction.

## Step variants

```tsx
<FlowStep id="start" label="Start" variant="circle" />
<FlowStep id="process" label="Process" variant="rectangle" />
<FlowStep id="decision" label="Decide" variant="diamond" />
<FlowStep id="end" label="End" variant="circle" />
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-surface` | Step background |
| `--s-border` | Step border and edge stroke |
| `--s-primary` | Active step accent |
| `--s-text` | Step label |
| `--s-radius-md` | Rectangle step radius |
| `--s-font-mono` | Label font |
| `--s-duration-slow` | Edge animation speed |

## Props

### FlowDiagram

| Prop | Type | Default | Description |
|---|---|---|---|
| `direction` | `"horizontal" \| "vertical"` | `"horizontal"` | Flow direction |
| `animated` | `boolean` | `false` | Animate edge flow |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Steps and branches |

### FlowStep

| Prop | Type | Default | Description |
|---|---|---|---|
| `id` | `string` | — | Unique step identifier |
| `label` | `string` | — | Step label |
| `variant` | `"rectangle" \| "circle" \| "diamond"` | `"rectangle"` | Step shape |

### FlowBranch

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | Branch condition label |
| `children` | `ReactNode` | — | Branch paths (2+ FlowSteps) |

## Accessibility

- SVG edges have `aria-hidden="true"`
- Steps are focusable with `role="listitem"` inside a `role="list"` container
