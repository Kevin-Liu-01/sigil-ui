title: StackDiagram
description: "A vertical stack diagram showing layered technology or architecture components."
---

## Import

```tsx
import { StackDiagram } from "@sigil-ui/components";
```

## Usage

```tsx
<StackDiagram
  layers={[
    { label: "Frontend", description: "React" },
    { label: "Backend", description: "Node.js" },
    { label: "Database", description: "PostgreSQL" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `layers` | `StackLayer[]` | — | Stack layers from top to bottom |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Layer borders |
| `--s-surface` | Layer background |
| `--s-text` | Layer label |
| `--s-text-muted` | Layer description |
| `--s-radius-md` | Layer border radius |
