## Import

```tsx
import { ProgressRing } from "@sigil-ui/components";
```

<ComponentPreview>

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `tracks` | `ProgressRingTrack[]` | ‚Äî | Ring tracks with value/max |
| `width` | `number` | `160` | SVG width |
| `height` | `number` | `160` | SVG height |
| `thickness` | `number` | `8` | Ring thickness |
| `gap` | `number` | `4` | Gap between rings |
| `showLegend` | `boolean` | `true` | Show legend |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-error` | First ring color |
| `--s-success` | Second ring color |
| `--s-info` | Third ring color |
| `--s-border` | Track background |
| `--s-text` | Legend percentage |
| `--s-text-muted` | Legend label |
| `--s-font-mono` | Percentage font |
