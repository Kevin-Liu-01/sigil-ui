## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `series` | `LineChartSeries[]` | — | Data series with points |
| `width` | `number` | `500` | SVG width |
| `height` | `number` | `250` | SVG height |
| `showDots` | `boolean` | `true` | Show dots at data points |
| `showGrid` | `boolean` | `true` | Show grid lines |
| `showXLabels` | `boolean` | `true` | Show x-axis labels |
| `showYLabels` | `boolean` | `true` | Show y-axis labels |
| `filled` | `boolean` | `false` | Fill area under line |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default line color |
| `--s-border` | Grid lines |
| `--s-surface` | Dot fill |
| `--s-text-muted` | Axis labels |
| `--s-font-mono` | Label font |
