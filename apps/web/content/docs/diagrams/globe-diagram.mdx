title: GlobeDiagram
description: "A stylized globe diagram showing geographic points or global coverage."
---

## Import

```tsx
import { GlobeDiagram } from "@sigil-ui/components";
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `cities` | `GlobeCity[]` | — | Array of cities with name, lat, lng |
| `width` | `number` | `300` | SVG width |
| `height` | `number` | `300` | SVG height |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | City dot color |
| `--s-border` | Globe outline and grid |
| `--s-text-muted` | City labels |
| `--s-font-mono` | Label font |
