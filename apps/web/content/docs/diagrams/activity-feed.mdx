## Import

```tsx
import { ActivityFeed } from "@sigil-ui/components";
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `entries` | `ActivityEntry[]` | — | Feed entries |
| `maxEntries` | `number` | — | Truncate after N entries |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-success` | Success dot color |
| `--s-warning` | Warning dot color |
| `--s-error` | Error dot color |
| `--s-info` | Info dot color |
| `--s-border` | Timeline line |
| `--s-text` | Entry title |
| `--s-text-muted` | Description and timestamp |
| `--s-font-mono` | Timestamp font |
