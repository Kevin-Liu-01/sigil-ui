## Import

```tsx
import { PieChart } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <PieChart
    slices={[
      { label: "Direct", value: 45 },

    ]}
  />

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `slices` | `PieSlice[]` | ‚Äî | Pie slices with label and value |
| `width` | `number` | `220` | SVG width |
| `height` | `number` | `220` | SVG height |
| `showLabels` | `boolean` | `true` | Show percentage labels on slices |
| `showLegend` | `boolean` | `true` | Show legend below chart |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | First slice color |
| `--s-success` | Second slice color |
| `--s-warning` | Third slice color |
| `--s-error` | Fourth slice color |
| `--s-text-muted` | Legend text |
