## Usage

```tsx
<PricingTable
  columns={[
    { name: "Free", price: "$0" },
    { name: "Pro", price: "$19", period: "mo", highlighted: true, badge: "Popular" },
    { name: "Enterprise", price: "Custom", period: "mo" },
  ]}
  features={[
    { name: "Users", values: { Free: "1", Pro: "10", Enterprise: "Unlimited" } },
    { name: "Storage", values: { Free: "1 GB", Pro: "100 GB", Enterprise: "Unlimited" } },
    { name: "API Access", values: { Free: false, Pro: true, Enterprise: true } },
    { name: "SSO", values: { Free: false, Pro: false, Enterprise: true } },
  ]}
  categories={[{ label: "Usage", startIndex: 0 }, { label: "Security", startIndex: 2 }]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `columns` | `PricingColumn[]` | — | Tier definitions with name, price, badge |
| `features` | `PricingFeature[]` | — | Feature rows with values per tier |
| `categories` | `{ label, startIndex }[]` | — | Optional category grouping headers |
