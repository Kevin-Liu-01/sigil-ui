---
title: CapabilityGrid
description: Animated capability chip matrix with category-by-category staggered reveals. Supports callout sections for comparison.
---

## Import

```tsx
import { CapabilityGrid } from "@sigil-ui/components";
```

## Usage

```tsx
<CapabilityGrid
  categories={[
    {
      label: "Runtimes",
      items: [
        { name: "Python", color: "var(--s-primary)" },
        { name: "Node.js", color: "var(--s-primary)" },
        { name: "Rust", color: "var(--s-primary)" },
      ],
    },
    {
      label: "Packages",
      items: [
        { name: "apt", color: "var(--s-warning)" },
        { name: "pip", color: "var(--s-warning)" },
      ],
    },
  ]}
  callout={{
    label: "Sandboxes restrict",
    items: ["No root access", "No GPU", "Read-only FS"],
    variant: "danger",
  }}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `categories` | `CapabilityCategory[]` | — | Chip categories |
| `callout` | `CapabilityCallout` | — | Comparison callout |

Categories and chips stagger into view on scroll.
