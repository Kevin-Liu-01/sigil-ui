---
title: ArchitectureDiagram
description: A system architecture diagram with labeled service boxes, connection arrows, and group boundaries. Built on the Diagram primitive.
---

## Import

```tsx
import { ArchitectureDiagram, Service, ServiceGroup, Connection } from "@sigil-ui/components";
```

## Usage

```tsx
<ArchitectureDiagram>
  <Service id="client" label="Client" x={50} y={100} />
  <Service id="api" label="API Server" x={250} y={100} />
  <Service id="db" label="Database" x={450} y={100} />
  <Connection from="client" to="api" label="HTTPS" />
  <Connection from="api" to="db" label="TCP" />
</ArchitectureDiagram>
```

## Service groups

```tsx
<ArchitectureDiagram>
  <ServiceGroup label="Frontend" x={20} y={20} width={200} height={180}>
    <Service id="web" label="Web App" x={50} y={60} />
    <Service id="mobile" label="Mobile" x={50} y={120} />
  </ServiceGroup>
  <ServiceGroup label="Backend" x={280} y={20} width={200} height={180}>
    <Service id="api" label="API" x={310} y={60} />
    <Service id="worker" label="Worker" x={310} y={120} />
  </ServiceGroup>
  <Connection from="web" to="api" />
  <Connection from="mobile" to="api" />
  <Connection from="api" to="worker" style="dashed" />
</ArchitectureDiagram>
```

## Service icons

```tsx
<Service id="db" label="PostgreSQL" icon={<DatabaseIcon />} x={100} y={100} />
<Service id="cache" label="Redis" icon={<ZapIcon />} x={300} y={100} />
```

## Connection styles

```tsx
<Connection from="a" to="b" style="solid" />
<Connection from="b" to="c" style="dashed" />
<Connection from="c" to="d" style="dotted" animated />
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-surface` | Service background |
| `--s-surface-elevated` | Group background |
| `--s-border` | Service border and connections |
| `--s-border-muted` | Group boundary |
| `--s-primary` | Selected service accent |
| `--s-text` | Label text |
| `--s-text-muted` | Connection labels |
| `--s-radius-md` | Service border radius |
| `--s-font-mono` | Label font |

## Props

### ArchitectureDiagram

| Prop | Type | Default | Description |
|---|---|---|---|
| `width` | `number` | `800` | Canvas width |
| `height` | `number` | `500` | Canvas height |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Services, groups, connections |

### Service

| Prop | Type | Default | Description |
|---|---|---|---|
| `id` | `string` | — | Unique service identifier |
| `label` | `string` | — | Service name |
| `icon` | `ReactNode` | — | Service icon |
| `x` | `number` | — | X position |
| `y` | `number` | — | Y position |

### Connection

| Prop | Type | Default | Description |
|---|---|---|---|
| `from` | `string` | — | Source service id |
| `to` | `string` | — | Target service id |
| `label` | `string` | — | Connection label |
| `style` | `"solid" \| "dashed" \| "dotted"` | `"solid"` | Line style |
| `animated` | `boolean` | `false` | Animate flow direction |

## Accessibility

- SVG canvas has `role="img"` with `aria-label` describing the architecture
- Services are focusable for keyboard exploration
