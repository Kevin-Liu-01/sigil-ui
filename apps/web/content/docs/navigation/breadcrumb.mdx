---
title: Breadcrumb
description: "A Breadcrumb component from the Sigil design system."
---

## Import

```tsx
import { Breadcrumb } from "@sigil-ui/components";
```

## Usage

```tsx
<Breadcrumb>Breadcrumb content</Breadcrumb>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `items` | `BreadcrumbItem[]` | — | Breadcrumb trail items. The last item is treated as the current page. |
| `separator` | `ReactNode` | `/` | Separator character. |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-text-muted` | Text muted |
| `--s-text` | Text |
| `--s-duration-fast` | Duration fast |
