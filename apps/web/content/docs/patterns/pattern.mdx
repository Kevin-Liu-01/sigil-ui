# Pattern

> Repeating SVG background patterns for decorative overlays — dots, grid, diagonal, diamond, hexagon, triangle, crosshatch.

## Installation

```bash
npx sigil add pattern
```

## Import

```tsx
import { Pattern } from "@sigil-ui/components";
```

## Usage

```tsx
<div className="relative h-64">
  <Pattern variant="dots" opacity={0.08} />
  <div className="relative z-10">Content on top</div>
</div>
```

## Examples

### Dots

```tsx
<Pattern variant="dots" opacity={0.1} />
```

### Crosshatch

```tsx
<Pattern variant="crosshatch" opacity={0.06} scale={1.5} />
```

### Diamond

```tsx
<Pattern variant="diamond" opacity={0.08} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `"dots" \| "hatch" \| "diagonal" \| "diamond" \| "hexagon" \| "triangle" \| "crosshatch"` | — | Pattern type |
| opacity | number | 0.1 | Pattern opacity |
| scale | number | 1 | Scale multiplier for pattern size |
| className | string | — | Additional classes |

## Tokens

- `--s-border` — pattern stroke color
