## Import

```tsx
import { Box3DGrid } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <Box3DGrid columns={3} />

## Usage

```tsx
<Box3DGrid>...</Box3DGrid>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `items` | `Box3DGridItem[]` | — | Items to display in the grid. |
| `columns` | `number` | `3` | Number of columns. |
| `gap` | `string | number` | `1.5rem` | Gap between boxes. |
