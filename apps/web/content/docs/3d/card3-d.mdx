## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `maxTilt` | `number` | `15` | Maximum tilt angle in degrees. |
| `scale` | `number` | `1.02` | Scale factor on hover. |
| `speed` | `number` | `400` | Transition speed in ms. |
