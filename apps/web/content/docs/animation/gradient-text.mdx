---
title: GradientText
description: Animated gradient text using background-clip. Shifts gradient colors smoothly for eye-catching headlines.
---

## Import

```tsx
import { GradientText } from "@sigil-ui/components";
```

## Usage

```tsx
<GradientText className="text-4xl font-bold">
  Sigil UI
</GradientText>
```

## Custom colors

```tsx
<GradientText
  colors={["#6366f1", "#ec4899", "#6366f1"]}
  duration={6}
>
  Custom gradient
</GradientText>
```

## Static (no animation)

```tsx
<GradientText animate={false}>Static gradient</GradientText>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `colors` | `string[]` | `["var(--s-primary)", "var(--s-accent)", "var(--s-primary)"]` | Gradient stops |
| `duration` | `number` | `4` | Animation cycle in seconds |
| `animate` | `boolean` | `true` | Enable animation |

Respects `prefers-reduced-motion`.
