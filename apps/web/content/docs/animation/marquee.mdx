---
title: Marquee
description: Infinite scrolling content in any direction. CSS-driven with pause-on-hover and configurable speed.
---

## Import

```tsx
import { Marquee } from "@sigil-ui/components";
```

## Usage

```tsx
<Marquee>
  <img src="/logo1.svg" alt="Logo 1" />
  <img src="/logo2.svg" alt="Logo 2" />
  <img src="/logo3.svg" alt="Logo 3" />
</Marquee>
```

## Directions

```tsx
<Marquee direction="left">Left (default)</Marquee>
<Marquee direction="right">Right</Marquee>
<Marquee direction="up">Up</Marquee>
<Marquee direction="down">Down</Marquee>
```

## Speed and gap

```tsx
<Marquee duration={15} gap="2rem">
  {logos.map((logo) => <img key={logo.alt} src={logo.src} alt={logo.alt} />)}
</Marquee>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `direction` | `"left" \| "right" \| "up" \| "down"` | `"left"` | Scroll direction |
| `duration` | `number` | `30` | Full cycle duration in seconds |
| `pauseOnHover` | `boolean` | `true` | Pause when hovered |
| `repeat` | `number` | `2` | Content copies for seamless loop |
| `gap` | `string` | `"1rem"` | Gap between items |

Respects `prefers-reduced-motion`.
