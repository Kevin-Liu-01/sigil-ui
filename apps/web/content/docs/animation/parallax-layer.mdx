## Usage

```tsx
<ParallaxLayer>...</ParallaxLayer>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `speed` | `number` | `0.3` | Parallax speed multiplier. Positive = moves slower than scroll, negative = faster. |
