---
title: WordRotate
description: Cycles through an array of words or phrases with blur + translate transitions. Inline-friendly for use within sentences.
---

## Import

```tsx
import { WordRotate } from "@sigil-ui/components";
```

## Usage

```tsx
<p>
  Build with <WordRotate words={["speed", "style", "confidence"]} />
</p>
```

## Custom duration

```tsx
<WordRotate words={["React", "Vue", "Svelte"]} duration={3000} />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `words` | `string[]` | — | Words to cycle through |
| `duration` | `number` | `2500` | Time each word is shown in ms |

Renders as inline `<span>`. Uses `aria-live="polite"` for screen readers.
