---
title: BlurFade
description: Modern blur + fade + translate reveal animation. Combines opacity, gaussian blur, and vertical offset for a polished entrance.
---

## Import

```tsx
import { BlurFade } from "@sigil-ui/components";
```

## Usage

```tsx
<BlurFade>
  <h1>Hero headline</h1>
</BlurFade>
```

## Custom blur and offset

```tsx
<BlurFade blur={16} yOffset={24} delay={300}>
  <p>Stronger blur, more offset</p>
</BlurFade>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `blur` | `number` | `10` | Initial blur radius in px |
| `yOffset` | `number` | `16` | Vertical translate in px |
| `delay` | `number` | `0` | Delay in ms |
| `trigger` | `"scroll" \| "mount"` | `"scroll"` | Trigger mode |
| `viewOptions` | `UseInViewOptions` | — | IntersectionObserver config |

Uses `--s-duration-slow` for a longer, more cinematic reveal. Respects `prefers-reduced-motion`.
