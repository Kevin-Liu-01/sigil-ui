## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `interval` | `number` | `80` | Delay between each child in ms. |
| `delay` | `number` | `0` | Base delay before the first child animates in ms. |
| `yOffset` | `number` | `20` | Vertical offset for each child. |
| `trigger` | `"scroll" | "mount"` | — | — |
| `viewOptions` | `UseInViewOptions` | — | — |
