title: TypeWriter
description: "A TypeWriter component from the Sigil design system."
---

## Import

```tsx
import { TypeWriter } from "@sigil-ui/components";
```

## Usage

```tsx
<TypeWriter>...</TypeWriter>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `words` | `string[]` | — | Text strings to type. Loops if multiple. |
| `speed` | `number` | `60` | Typing speed in ms per character. |
| `pauseDuration` | `number` | `1500` | Pause after a word is fully typed in ms. |
| `deleteSpeed` | `number` | `40` | Deleting speed in ms per character. |
| `loop` | `boolean` | `true` | Whether to loop through words. |
| `cursor` | `boolean` | `true` | Show blinking cursor. |
