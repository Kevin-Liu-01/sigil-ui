---
title: ChangelogTable
description: "A versioned changelog table with typed entry badges for release notes."
---

## Import

```tsx
import { ChangelogTable } from "@sigil-ui/components";
```

## Usage

```tsx
<ChangelogTable
  releases={[
    { version: "2.1.0", date: "2026-04-20", entries: [...] },
  ]}
  showBadges
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `releases` | `ChangelogRelease[]` | — | Release objects with version, date, and entries |
| `showBadges` | `boolean` | `true` | Show colored type badges (added/fixed/changed/etc.) |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-success` | "added" badge |
| `--s-info` | "changed" badge |
| `--s-warning` | "fixed" badge |
| `--s-error` | "removed" and "security" badges |
| `--s-text-muted` | "deprecated" badge and date |
| `--s-border` | Card and divider borders |
| `--s-surface` | Card background |
| `--s-surface-elevated` | Version header background |
| `--s-text` | Entry text and version |
| `--s-font-mono` | Version number font |
| `--s-radius-card` | Card radius |
| `--s-radius-sm` | Badge radius |
