## Import

```tsx
import { Listbox } from "@sigil-ui/components";
```

## Usage

```tsx
<Listbox value="modal" options={[{ value: "modal", label: "Modal" }, { value: "sheet", label: "Sheet" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add listbox
```

## Token Integration

Listbox follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
