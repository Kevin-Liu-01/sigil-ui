# CodeBlock

> Syntax-highlighted code display with line numbers, file name, and copy button.

## Installation

```bash
npx @sigil-ui/cli add code-block
```

## Import

```tsx
import { CodeBlock } from "@sigil-ui/components";
```

## Usage

```tsx
<CodeBlock
  code={`const greeting = "Hello, Sigil";
console.log(greeting);`}
  language="typescript"
/>
```

## Examples

### With file name

```tsx
<CodeBlock
  code={`export default function App() {
  return <h1>Welcome</h1>;
}`}
  language="tsx"
  fileName="app/page.tsx"
/>
```

### Without line numbers

```tsx
<CodeBlock
  code={`npm install @sigil-ui/components`}
  language="bash"
  showLineNumbers={false}
/>
```

### Multi-line with scroll

```tsx
<CodeBlock
  code={longCodeString}
  language="typescript"
  fileName="utils/tokens.ts"
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `code` | `string` | — | Code content to display |
| `language` | `string` | — | Language label shown in header |
| `showLineNumbers` | `boolean` | `true` | Show line numbers |
| `fileName` | `string` | — | File name shown in header (overrides language label) |
| `className` | `string` | — | Additional CSS classes |

## Tokens

This component reads these CSS variables:

- `--s-background` — code area background
- `--s-surface` — header bar background
- `--s-border` — outer border and header divider
- `--s-card-radius` — container border radius
- `--s-text-muted` — line numbers, header labels, copy button
- `--s-text` — copy button hover
