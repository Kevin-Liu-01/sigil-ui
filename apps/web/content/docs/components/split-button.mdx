---
title: SplitButton
description: Button with a primary action and a separate dropdown trigger, sharing one visual container.
---

## Import

```tsx
import { SplitButton } from "@sigil-ui/components";
```

## Usage

```tsx
<SplitButton onClick={() => console.log("save")}>
  Save
</SplitButton>
```

## With icon

```tsx
<SplitButton
  icon={<PlusIcon className="size-4" />}
  onClick={() => console.log("create")}
  onDropdownClick={() => console.log("dropdown")}
>
  Create
</SplitButton>
```

## Custom dropdown handler

```tsx
<SplitButton
  onClick={() => alert("Primary action")}
  onDropdownClick={() => alert("Show menu")}
  dropdownLabel="Show more actions"
>
  Deploy
</SplitButton>
```

## Disabled

```tsx
<SplitButton disabled>Unavailable</SplitButton>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `children` | `ReactNode` | — | Button label |
| `icon` | `ReactNode` | — | Leading icon |
| `onClick` | `function` | — | Primary button click handler |
| `onDropdownClick` | `function` | — | Dropdown trigger click handler |
| `dropdownLabel` | `string` | `"More options"` | Accessible label for dropdown button |
| `disabled` | `boolean` | `false` | Disable both buttons |
| `className` | `string` | — | Additional CSS classes |
