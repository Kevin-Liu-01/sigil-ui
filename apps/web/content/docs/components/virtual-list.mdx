## Import

```tsx
import { VirtualList } from "@sigil-ui/components";
```

```tsx
<VirtualList items={["Modal", "MultiSelect", "StatusBadge", "DataToolbar"]} />
```

## CLI

```bash
npx @sigil-ui/cli add virtual-list
```

## Token Integration

VirtualList follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
