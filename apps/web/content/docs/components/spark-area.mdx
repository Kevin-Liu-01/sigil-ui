## Import

```tsx
import { SparkArea } from "@sigil-ui/components";
```

## Usage

```tsx
<SparkArea values={[4, 8, 6, 12, 10, 16]} />
```

## CLI

```bash
npx @sigil-ui/cli add spark-area
```

## Token Integration

SparkArea follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
