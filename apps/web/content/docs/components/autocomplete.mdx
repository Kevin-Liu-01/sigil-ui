## Import

```tsx
import { Autocomplete } from "@sigil-ui/components";
```

## Usage

```tsx
<Autocomplete options={[{ value: "modal", label: "Modal" }, { value: "table", label: "DataTable" }]} placeholder="Pick a component" />
```

## CLI

```bash
npx @sigil-ui/cli add autocomplete
```

## Token Integration

Autocomplete follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
