## Import

```tsx
import { TimePicker } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <TimePicker defaultValue="09:30" />

## Usage

```tsx
<TimePicker defaultValue="09:30" />
```

## CLI

```bash
npx @sigil-ui/cli add time-picker
```

## Token Integration

TimePicker follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
