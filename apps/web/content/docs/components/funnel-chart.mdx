## Usage

```tsx
<FunnelChart
  steps={[
    { label: "Visitors", value: 10000 },
    { label: "Signups", value: 3200 },
    { label: "Active", value: 1800 },
    { label: "Paid", value: 450 },
  ]}
  showRates
  showValues
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `steps` | `FunnelStep[]` | â€” | Funnel stages from widest to narrowest |
