## Import

```tsx
import { ResourceCard } from "@sigil-ui/components";
```

## Usage

```tsx
<ResourceCard title="Resource" description="A linked documentation resource." href="/docs" />
```

## CLI

```bash
npx @sigil-ui/cli add resource-card
```

## Token Integration

ResourceCard follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
