## Import

```tsx
import { DataToolbar } from "@sigil-ui/components";
```

## Preview

## Usage

```tsx
<DataToolbar><DataFilters searchPlaceholder="Filter..." /><Button variant="outline">Export</Button></DataToolbar>
```

## CLI

```bash
npx @sigil-ui/cli add data-toolbar
```

## Token Integration

DataToolbar follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
