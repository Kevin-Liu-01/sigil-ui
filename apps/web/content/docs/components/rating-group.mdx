---
title: RatingGroup
description: Star-based rating input with hover preview, keyboard support, and configurable size and count.
---

## Import

```tsx
import { RatingGroup } from "@sigil-ui/components";
```

## Usage

```tsx
<RatingGroup defaultValue={3} />
```

## Controlled

```tsx
import { useState } from "react";

function ProductRating() {
  const [rating, setRating] = useState(0);

  return <RatingGroup value={rating} onChange={setRating} />;
}
```

## Sizes

```tsx
<RatingGroup size="sm" defaultValue={4} />
<RatingGroup size="md" defaultValue={4} />
<RatingGroup size="lg" defaultValue={4} />
```

## Custom max

```tsx
<RatingGroup max={10} defaultValue={7} />
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-warning` | Filled star color (falls back to `#f59e0b`) |
| `--s-border` | Empty star color |
| `--s-primary` | Focus ring |
| `--s-radius-sm` | Focus outline radius |
| `--s-duration-fast` | Hover color transition |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | — | Controlled rating value |
| `defaultValue` | `number` | `0` | Initial rating (uncontrolled) |
| `onChange` | `(value: number) => void` | — | Called when rating changes |
| `max` | `number` | `5` | Number of stars |
| `disabled` | `boolean` | `false` | Disable interaction |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Star icon size |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Uses `role="radiogroup"` with individual `role="radio"` stars
- Each star has `aria-checked` and an `aria-label` like "3 stars"
- Hover preview shows visual feedback before committing
