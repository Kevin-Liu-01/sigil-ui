---
title: InputOTP
description: A one-time password input with auto-focus, paste support, and token-driven styling for verification flows.
---

## Import

```tsx
import { InputOTP, InputOTPGroup, InputOTPSlot, InputOTPSeparator } from "@sigil-ui/components";
```

## Usage

```tsx
<InputOTP maxLength={6}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
  </InputOTPGroup>
  <InputOTPSeparator />
  <InputOTPGroup>
    <InputOTPSlot index={3} />
    <InputOTPSlot index={4} />
    <InputOTPSlot index={5} />
  </InputOTPGroup>
</InputOTP>
```

## Four-digit code

```tsx
<InputOTP maxLength={4}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
    <InputOTPSlot index={3} />
  </InputOTPGroup>
</InputOTP>
```

## Controlled

```tsx
import { useState } from "react";

function VerifyCode() {
  const [value, setValue] = useState("");

  return (
    <InputOTP maxLength={6} value={value} onChange={setValue}>
      <InputOTPGroup>
        <InputOTPSlot index={0} />
        <InputOTPSlot index={1} />
        <InputOTPSlot index={2} />
        <InputOTPSlot index={3} />
        <InputOTPSlot index={4} />
        <InputOTPSlot index={5} />
      </InputOTPGroup>
    </InputOTP>
  );
}
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-surface` | Slot background |
| `--s-border` | Slot border |
| `--s-border-interactive` | Focused slot border |
| `--s-text` | Digit color |
| `--s-radius-md` | Slot border radius |
| `--s-duration-fast` | Focus transition |

## Props

### InputOTP

| Prop | Type | Default | Description |
|---|---|---|---|
| `maxLength` | `number` | — | Total number of digits |
| `value` | `string` | — | Controlled value |
| `onChange` | `(value: string) => void` | — | Change handler |
| `disabled` | `boolean` | `false` | Disable all slots |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Each slot is an `input` with `inputMode="numeric"` and `autocomplete="one-time-code"`
- Focus auto-advances to the next slot on input
- Paste fills all slots at once
