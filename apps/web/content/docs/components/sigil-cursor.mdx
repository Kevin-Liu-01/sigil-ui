## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `enabled` | `boolean` | `true` | Enable the custom cursor. |
| `hideNative` | `boolean` | `true` | Hide the native cursor while this component is mounted. |
| `variant` | `SigilCursorVariant` | `sigil` | Cursor drawing style. |
| `size` | `number | string` | ‚Äî | Cursor outer size. Accepts a number of pixels or any CSS length. |
| `dotSize` | `number | string` | ‚Äî | Center dot size. Accepts a number of pixels or any CSS length. |
| `color` | `string` | ‚Äî | Primary cursor color. |
| `ringColor` | `string` | ‚Äî | Ring stroke color. |
| `dotColor` | `string` | ‚Äî | Center dot color. |
| `blendMode` | `CSSProperties["mixBlendMode"]` | ‚Äî | CSS mix-blend-mode value. |
