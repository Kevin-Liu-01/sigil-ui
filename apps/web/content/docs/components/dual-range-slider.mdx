## Import

```tsx
import { DualRangeSlider } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>

## Usage

```tsx
<DualRangeSlider defaultValue={[20, 80]} />
```

## CLI

```bash
npx @sigil-ui/cli add dual-range-slider
```

## Token Integration

DualRangeSlider follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
