
```tsx
<div className="relative h-32 rounded-[var(--s-card-radius)] border border-[color:var(--s-border)]"><SpinnerOverlay /></div>
```

## CLI

```bash
npx @sigil-ui/cli add spinner-overlay
```

## Token Integration

SpinnerOverlay follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
