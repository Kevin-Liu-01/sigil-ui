
```tsx
<CodeTabs tabs={[{ value: "tsx", label: "TSX", code: "<Button>Ship</Button>", language: "tsx" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add code-tabs
```

## Token Integration

CodeTabs follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
