## Import

```tsx
import { ActionMenu } from "@sigil-ui/components";
```

## Usage

```tsx
<ActionMenu trigger={<Button variant="outline">Actions</Button>} items={[{ label: "Copy" }, { label: "Duplicate" }, { label: "Archive" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add action-menu
```

## Token Integration

ActionMenu follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
