## Import

```tsx
import { ComboboxField } from "@sigil-ui/components";
```

```tsx
<ComboboxField label="Component" options={[{ value: "modal", label: "Modal" }, { value: "select", label: "Select" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add combobox-field
```

## Token Integration

ComboboxField follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
