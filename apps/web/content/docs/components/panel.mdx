# Panel

> Bordered content container with token-driven surface, border, and radius. Supports compact and square layout modes.

## Installation

```bash
npx @sigil-ui/cli add panel
```

## Import

```tsx
import { Panel, PanelHead } from "@sigil-ui/components";
```

## Usage

```tsx
<Panel>
  <PanelHead>
    <h3>Settings</h3>
    <Button size="sm" variant="outline">Save</Button>
  </PanelHead>
  <p>Panel content goes here.</p>
</Panel>
```

## Examples

### Compact

```tsx
<Panel compact>
  <PanelHead>
    <span>Compact panel</span>
  </PanelHead>
  <p>Tighter padding for dense layouts.</p>
</Panel>
```

### Square

Flush corners for inset layouts inside an AppShell or detail page.

```tsx
<Panel square>
  <PanelHead>
    <span>Detail View</span>
  </PanelHead>
  <p>Square variant removes border radius.</p>
</Panel>
```

### Square + Compact

```tsx
<Panel square compact>
  <p>Tightest rhythm for tool panels.</p>
</Panel>
```

## Props

### Panel

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `compact` | `boolean` | `false` | Tighter padding rhythm |
| `square` | `boolean` | `false` | Flush corners for inset layouts |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Panel content |

### PanelHead

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Header content |

## Tokens

- `--s-border` — panel border color
- `--s-border-style` — border style (solid, dashed, etc.)
- `--s-surface` — panel background
- `--s-card-radius` — border radius (when not `square`)
