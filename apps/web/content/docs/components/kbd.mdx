---
title: Kbd
description: Keyboard shortcut indicator with optional grouping for multi-key combos.
---

## Import

```tsx
import { Kbd, KbdGroup } from "@sigil-ui/components";
```

## Usage

```tsx
<Kbd>⌘</Kbd>
```

## Key combination

```tsx
<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```

## In context

```tsx
<p className="text-sm text-[var(--s-text-muted)]">
  Press <KbdGroup><Kbd>⌘</Kbd><Kbd>S</Kbd></KbdGroup> to save
</p>
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-radius-sm` | Key border radius |
| `--s-border` | Key border |
| `--s-border-style` | Border style |
| `--s-surface` | Key background |
| `--s-text-muted` | Key text color |

## Props

### Kbd

Extends `HTMLAttributes<HTMLElement>`. Renders a `<kbd>` element styled as a physical key.

### KbdGroup

Extends `HTMLAttributes<HTMLDivElement>`. Inline flex container that spaces multiple `<Kbd>` elements with a small gap.
