## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | — | Progress value (0–100). Omit for indeterminate. |
| `label` | `boolean` | — | Show percentage label. |
| `size` | `"sm" | "md" | "lg"` | `md` | — |
