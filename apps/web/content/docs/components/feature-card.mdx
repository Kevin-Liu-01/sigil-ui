## Import

```tsx
import { FeatureCard } from "@sigil-ui/components";
```

## Usage

```tsx
<FeatureCard title="Feature" description="A feature summary card." />
```

## CLI

```bash
npx @sigil-ui/cli add feature-card
```

## Token Integration

FeatureCard follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
