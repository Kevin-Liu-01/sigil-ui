## CLI

```bash
npx @sigil-ui/cli add tree-table
```

## Token Integration

TreeTable follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
