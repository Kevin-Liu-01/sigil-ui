## CLI

```bash
npx @sigil-ui/cli add tooltip-group
```

## Token Integration

TooltipGroup follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
