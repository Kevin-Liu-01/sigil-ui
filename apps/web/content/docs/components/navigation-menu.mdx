---
title: NavigationMenu
description: A site-wide navigation component with dropdown panels, viewport animations, and responsive collapse. Built on Radix NavigationMenu.
---

## Import

```tsx
import {
  NavigationMenu, NavigationMenuList, NavigationMenuItem,
  NavigationMenuTrigger, NavigationMenuContent, NavigationMenuLink,
} from "@sigil-ui/components";
```

## Usage

```tsx
<NavigationMenu>
  <NavigationMenuList>
    <NavigationMenuItem>
      <NavigationMenuTrigger>Products</NavigationMenuTrigger>
      <NavigationMenuContent>
        <Stack gap={8} className="p-4 w-[400px]">
          <NavigationMenuLink href="/components">Components</NavigationMenuLink>
          <NavigationMenuLink href="/presets">Presets</NavigationMenuLink>
        </Stack>
      </NavigationMenuContent>
    </NavigationMenuItem>
    <NavigationMenuItem>
      <NavigationMenuLink href="/docs">Docs</NavigationMenuLink>
    </NavigationMenuItem>
  </NavigationMenuList>
</NavigationMenu>
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-surface` | Content panel background |
| `--s-surface-elevated` | Link hover background |
| `--s-text` | Link text |
| `--s-text-muted` | Description text |
| `--s-border` | Panel border |
| `--s-radius-lg` | Panel border radius |
| `--s-shadow-lg` | Panel elevation |
| `--s-duration-normal` | Viewport animation |
| `--s-ease-default` | Animation easing |

## Props

### NavigationMenuTrigger

| Prop | Type | Default | Description |
|---|---|---|---|
| `className` | `string` | тАФ | Additional CSS classes |
| `children` | `ReactNode` | тАФ | Trigger label |

### NavigationMenuLink

| Prop | Type | Default | Description |
|---|---|---|---|
| `href` | `string` | тАФ | Link destination |
| `active` | `boolean` | `false` | Active page indicator |
| `className` | `string` | тАФ | Additional CSS classes |
| `children` | `ReactNode` | тАФ | Link content |

## Accessibility

- `role="navigation"` with `aria-label`
- Arrow key navigation between menu items
- Escape closes open panels and returns focus to trigger
