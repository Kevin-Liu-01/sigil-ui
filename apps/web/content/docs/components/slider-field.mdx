---

## Import

```tsx
import { SliderField } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <SliderField label="Volume" valueLabel="40%" defaultValue={[40]} />
</ComponentPreview>

## Usage

```tsx
<SliderField label="Volume" valueLabel="40%" defaultValue={[40]} />
```

## CLI

```bash
npx @sigil-ui/cli add slider-field
```

## Token Integration

SliderField follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
