## Import

```tsx
import { DataGrid } from "@sigil-ui/components";
```

## Preview

## Usage

```tsx
<DataGrid columns={2}><StatCard label="A" value="12" /><StatCard label="B" value="24" /></DataGrid>
```

## CLI

```bash
npx @sigil-ui/cli add data-grid
```

## Token Integration

DataGrid follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
