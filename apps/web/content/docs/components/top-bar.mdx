## Import

```tsx
import { TopBar } from "@sigil-ui/components";
```

## Usage

```tsx
<TopBar>Top bar content</TopBar>
```

## CLI

```bash
npx @sigil-ui/cli add top-bar
```

## Token Integration

TopBar follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
