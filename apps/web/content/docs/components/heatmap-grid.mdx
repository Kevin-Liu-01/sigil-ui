## Import

```tsx
import { HeatmapGrid } from "@sigil-ui/components";
```

## Usage

```tsx
<HeatmapGrid
  xLabels={["Mon", "Tue", "Wed", "Thu", "Fri"]}
  yLabels={["9am", "12pm", "3pm", "6pm"]}
  data={matrix}
  cellSize={40}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `HeatmapCell[][]` | ‚Äî | 2D array of cell values |
| `xLabels` | `string[]` | ‚Äî | Column labels |
| `yLabels` | `string[]` | ‚Äî | Row labels |
| `min` | `number` | auto | Minimum value for color scale |
| `max` | `number` | auto | Maximum value for color scale |
| `color` | `string` | `var(--s-primary)` | High-intensity cell color |
| `cellSize` | `number` | `40` | Cell size in px |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default cell color |
| `--s-primary-contrast` | Cell text color |
| `--s-text-muted` | Axis labels |
| `--s-font-mono` | Label and value font |
| `--s-radius-sm` | Cell corner radius |
