---
title: Aside
description: Sidebar panel with configurable width, optional sticky positioning, and border control.
---

## Import

```tsx
import { Aside } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <Aside>This is supplementary content placed alongside the main text.</Aside>
</ComponentPreview>

## Usage

```tsx
<Aside>
  <nav>Sidebar navigation</nav>
</Aside>
```

## Custom width and border

```tsx
<Aside width={320} borderSide="left">
  Right-placed sidebar with left border
</Aside>
```

## Non-sticky

```tsx
<Aside sticky={false} borderSide="none">
  Scrolls with content, no border
</Aside>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `width` | `number \| string` | `256` | Sidebar width (number = px) |
| `sticky` | `boolean` | `true` | Stick to viewport top with `position: sticky` |
| `borderSide` | `"left" \| "right" \| "none"` | `"right"` | Which side to show a border |
| `className` | `string` | — | Additional CSS classes |
