---
title: MermaidDiagram
description: Renders Mermaid chart syntax as an SVG diagram with dark and default theme support.
---

## Import

```tsx
import { MermaidDiagram } from "@sigil-ui/components";
```

## Preview

<ComponentPreview vertical>
  <MermaidDiagram
    chart={`graph LR
      A[Tokens] --> B[CSS]
      B --> C[Components]
      C --> D[App]`}
  />
</ComponentPreview>

## Usage

```tsx
<MermaidDiagram
  chart={`graph LR
    A[Request] --> B[Auth]
    B --> C[Handler]
    C --> D[Response]`}
/>
```

## Default theme

```tsx
<MermaidDiagram
  theme="default"
  chart={`sequenceDiagram
    Client->>Server: POST /login
    Server-->>Client: 200 OK`}
/>
```

## Flowchart

```tsx
<MermaidDiagram
  chart={`flowchart TD
    Start --> Check{Valid?}
    Check -- Yes --> Process
    Check -- No --> Error
    Process --> Done`}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `chart` | `string` | — | Mermaid chart definition string (required) |
| `theme` | `"dark" \| "default"` | `"dark"` | Mermaid rendering theme |
| `className` | `string` | — | Additional CSS classes |

Plus all standard `<div>` HTML attributes.

Requires `mermaid` as a peer dependency — it is dynamically imported at render time.
