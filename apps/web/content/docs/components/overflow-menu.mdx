```tsx
import { OverflowMenu } from "@sigil-ui/components";
```

```tsx
<OverflowMenu items={[{ label: "Edit" }, { label: "Share" }, { label: "Delete" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add overflow-menu
```

## Token Integration

OverflowMenu follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
