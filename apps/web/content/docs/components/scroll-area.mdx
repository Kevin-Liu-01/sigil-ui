---
title: ScrollArea
description: A custom scrollbar container built on Radix ScrollArea. Token-driven scrollbar colors, width, and radius.
---

## Import

```tsx
import { ScrollArea } from "@sigil-ui/components";
```

## Usage

```tsx
<ScrollArea className="h-[200px]">
  <div className="p-4">
    {items.map((item) => (
      <p key={item.id}>{item.text}</p>
    ))}
  </div>
</ScrollArea>
```

## Horizontal scrolling

```tsx
<ScrollArea orientation="horizontal" className="w-full">
  <Stack direction="row" gap={16} className="p-4 w-max">
    {cards.map((card) => <Card key={card.id}>{card.title}</Card>)}
  </Stack>
</ScrollArea>
```

## Both axes

```tsx
<ScrollArea orientation="both" className="h-[300px] w-[400px]">
  <div className="w-[800px] h-[600px] p-4">Large content area</div>
</ScrollArea>
```

## Custom thickness

```tsx
<ScrollArea scrollbarSize={14} className="h-[240px]">
  <div className="p-4">Chunkier scrollbar, same token system.</div>
</ScrollArea>
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-scrollbar-width` | Vertical scrollbar width |
| `--s-scrollbar-height` | Horizontal scrollbar height |
| `--s-scrollbar-padding` | Internal track padding |
| `--s-scrollbar-track` | Track background |
| `--s-scrollbar-thumb` | Thumb color |
| `--s-scrollbar-thumb-hover` | Thumb hover color |
| `--s-scrollbar-thumb-active` | Thumb active color |
| `--s-scrollbar-radius` | Track and thumb radius |
| `--s-duration-fast` | Thumb fade transition |

Native scroll containers can opt into the same system with `className="sigil-scrollbar"` or `data-sigil-scrollbar`.

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `orientation` | `"vertical" \| "horizontal" \| "both"` | `"vertical"` | Scroll direction |
| `scrollbarSize` | `number \| string` | token value | Scrollbar thickness for this instance |
| `scrollbarClassName` | `string` | — | Additional classes for the track |
| `thumbClassName` | `string` | — | Additional classes for the thumb |
| `className` | `string` | — | Additional CSS classes |
| `children` | `ReactNode` | — | Scrollable content |

## Accessibility

- Scrollbar is styled only — native scroll behavior is preserved
- Keyboard scrolling works natively with arrow keys and Page Up/Down
- Scrollbar is visible on hover and during scroll, hidden otherwise
