# Password Input

> Standard password input with a toggle button to reveal or hide the password text.

## Installation

```bash
npx sigil add password-input
```

## Import

```tsx
import { PasswordInput } from "@sigil-ui/components";
```

## Usage

```tsx
<PasswordInput placeholder="Enter password" />
```

## Examples

### With label

```tsx
<div className="flex flex-col gap-2">
  <Label htmlFor="password">Password</Label>
  <PasswordInput id="password" placeholder="••••••••" />
</div>
```

### Without toggle

```tsx
<PasswordInput placeholder="Secret key" showToggle={false} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `showToggle` | `boolean` | `true` | Show the visibility toggle button |
| `placeholder` | `string` | — | Input placeholder text |
| `className` | `string` | — | Additional CSS classes |

All standard `<input>` props (except `type`) are also accepted.

## Tokens

- `--s-border` — input border color
- `--s-border-style` — border style
- `--s-background` — input background
- `--s-text` / `--s-text-muted` — text and placeholder colors
- `--s-input-height` — input height
- `--s-radius-md` — border radius
- `--s-ring` / `--s-primary` — focus ring
- `--s-duration-fast` — toggle transition speed
