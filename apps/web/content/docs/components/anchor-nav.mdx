```tsx
import { AnchorNav } from "@sigil-ui/components";
```

```tsx

```

## CLI

```bash
npx @sigil-ui/cli add anchor-nav
```

## Token Integration

AnchorNav follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
