---
title: Switch
description: A toggle switch for boolean settings. Built on Radix Switch with token-driven track and thumb colors.
---

## Import

```tsx
import { Switch } from "@sigil-ui/components";
```

## Usage

```tsx
<Switch />
```

## With label

```tsx
<Stack direction="row" gap={8} align="center">
  <Switch id="dark-mode" />
  <Label htmlFor="dark-mode">Dark mode</Label>
</Stack>
```

## Controlled

```tsx
import { useState } from "react";

function NotificationToggle() {
  const [enabled, setEnabled] = useState(false);

  return (
    <Stack direction="row" gap={8} align="center">
      <Switch checked={enabled} onCheckedChange={setEnabled} />
      <Label>Notifications {enabled ? "on" : "off"}</Label>
    </Stack>
  );
}
```

## Sizes

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-primary` | Checked track color |
| `--s-surface` | Unchecked track color |
| `--s-border` | Track border |
| `--s-text-disabled` | Disabled state |
| `--s-radius-full` | Track and thumb radius |
| `--s-duration-fast` | Toggle transition |
| `--s-ease-spring` | Thumb slide easing |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `checked` | `boolean` | — | Controlled checked state |
| `defaultChecked` | `boolean` | `false` | Default checked state |
| `onCheckedChange` | `(checked: boolean) => void` | — | Change handler |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Switch size |
| `disabled` | `boolean` | `false` | Disable interactions |
| `className` | `string` | — | Additional CSS classes |

## Accessibility

- Uses `role="switch"` with `aria-checked`
- Toggles on Space and Enter
- Focus ring uses `--s-border-interactive`
