## Import

```tsx
import { PropertyList } from "@sigil-ui/components";
```

```tsx
<PropertyList items={[{ label: "Status", value: "Ready" }, { label: "Components", value: "300+" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add property-list
```

## Token Integration

PropertyList follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
