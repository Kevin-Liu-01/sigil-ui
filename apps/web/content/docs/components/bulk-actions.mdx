```tsx
import { BulkActions } from "@sigil-ui/components";
```

## Usage

```tsx
<BulkActions><span className="text-sm">3 selected</span><Button size="sm" variant="outline">Archive</Button></BulkActions>
```

## CLI

```bash
npx @sigil-ui/cli add bulk-actions
```

## Token Integration

BulkActions follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
