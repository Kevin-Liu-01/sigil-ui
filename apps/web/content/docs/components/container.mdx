---
title: Container
description: Max-width wrapper that centers page content with responsive padding.
---

## Import

```tsx
import { Container } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <Container className="border border-dashed border-[var(--s-border)] p-6">Content within a max-width container</Container>
</ComponentPreview>

## Usage

```tsx
<Container>
  <p>Content constrained to the default max-width.</p>
</Container>
```

## Size variants

```tsx
<Container size="narrow">Narrow (680px)</Container>
<Container size="default">Default (1200px)</Container>
<Container size="wide">Wide (1400px)</Container>
```

## Disable centering

```tsx
<Container center={false}>
  Left-aligned container
</Container>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `"narrow" \| "default" \| "wide"` | `"default"` | Max-width variant |
| `center` | `boolean` | `true` | Center the container with auto margins |
| `className` | `string` | — | Additional CSS classes |
