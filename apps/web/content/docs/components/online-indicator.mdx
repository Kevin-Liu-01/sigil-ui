## Import

```tsx
import { OnlineIndicator } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <OnlineIndicator status="online" label="Online" />
</ComponentPreview>

## Usage

```tsx
<OnlineIndicator status="online" label="Online" />
```

## CLI

```bash
npx @sigil-ui/cli add online-indicator
```

## Token Integration

OnlineIndicator follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
