## Import

```tsx
import { ThemeSwitcher } from "@sigil-ui/components";
```

## Usage

```tsx
<ThemeSwitcher value="sigil" themes={[{ value: "sigil", label: "Sigil" }, { value: "mono", label: "Mono" }]} />
```

## CLI

```bash
npx @sigil-ui/cli add theme-switcher
```

## Token Integration

ThemeSwitcher follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
