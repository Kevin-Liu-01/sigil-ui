
## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | — | Progress value between 0 and 100. Omit for indeterminate. |
| `size` | `number` | `40` | Diameter in pixels. |
| `strokeWidth` | `number` | `4` | Stroke width in pixels. |
