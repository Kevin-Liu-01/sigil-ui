## Import

```tsx
import { PhoneInput } from "@sigil-ui/components";
```

<ComponentPreview>

## Usage

```tsx
<PhoneInput placeholder="+1 (555) 000-0000" />
```

## CLI

```bash
npx @sigil-ui/cli add phone-input
```

## Token Integration

PhoneInput follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
