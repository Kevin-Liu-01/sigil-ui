---
title: Textarea
description: A multi-line text input with auto-resize, character count, and token-driven styling.
---

## Import

```tsx
import { Textarea } from "@sigil-ui/components";
```

## Usage

```tsx
<Textarea placeholder="Write your message..." />
```

## With label

```tsx
<Stack gap={8}>
  <Label htmlFor="bio">Bio</Label>
  <Textarea id="bio" placeholder="Tell us about yourself" />
</Stack>
```

## Auto-resize

The textarea grows with content up to an optional max height:

```tsx
<Textarea autoResize placeholder="Grows as you type..." />
<Textarea autoResize maxRows={8} placeholder="Grows up to 8 rows" />
```

## Character count

```tsx
<Textarea maxLength={280} showCount placeholder="Write a tweet..." />
```

## Variants

```tsx
<Textarea variant="outline" placeholder="Bordered" />
<Textarea variant="filled" placeholder="Filled background" />
<Textarea variant="flush" placeholder="Bottom border only" />
```

## Error state

```tsx
<Textarea error errorMessage="Message is required." />
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-surface` | Filled variant background |
| `--s-text` | Input text color |
| `--s-text-muted` | Placeholder and character count |
| `--s-border` | Default border |
| `--s-border-interactive` | Focus border |
| `--s-error` | Error state border |
| `--s-radius-md` | Border radius |
| `--s-font-body` | Font family |
| `--s-duration-fast` | Focus transition |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `"outline" \| "filled" \| "flush"` | `"outline"` | Visual variant |
| `autoResize` | `boolean` | `false` | Auto-grow with content |
| `maxRows` | `number` | — | Max rows for auto-resize |
| `maxLength` | `number` | — | Maximum characters |
| `showCount` | `boolean` | `false` | Display character count |
| `error` | `boolean` | `false` | Error state |
| `errorMessage` | `string` | — | Error text below textarea |
| `className` | `string` | — | Additional CSS classes |

Plus all standard `<textarea>` HTML attributes.

## Accessibility

- Uses native `<textarea>` element
- Error state adds `aria-invalid="true"` and `aria-describedby`
- Character count is announced via `aria-live="polite"`
