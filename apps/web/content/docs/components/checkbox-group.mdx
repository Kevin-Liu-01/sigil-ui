## Import

```tsx
import { CheckboxGroup, CheckboxGroupItem } from "@sigil-ui/components";
```

## Usage

```tsx
<CheckboxGroup defaultValue={["a"]} onValueChange={(value) => console.log(value)}>
  <CheckboxGroupItem value="a" label="Option A" description="Short explainer." />
  <CheckboxGroupItem value="b" label="Option B" />
</CheckboxGroup>
```

You can also pass an `items` array as a shorthand:

```tsx
<CheckboxGroup
  items={[
    { value: "a", label: "Option A" },
    { value: "b", label: "Option B" },
  ]}
/>
```

## Props

### CheckboxGroup

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `string[]` | ‚Äî | Controlled selection. |
| `defaultValue` | `string[]` | `[]` | Uncontrolled initial selection. |
| `onValueChange` | `(value: string[]) => void` | ‚Äî | Called with the next selection on every toggle. |
| `disabled` | `boolean` | `false` | Disables every item in the group. |
| `name` | `string` | ‚Äî | Form field name applied to each underlying checkbox. |
| `items` | `CheckboxGroupItemConfig[]` | ‚Äî | Shorthand for rendering items without children. |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | Stacks items in a column or wraps them in a row. |

### CheckboxGroupItem

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `string` | ‚Äî | Item value used to identify the selection. |
| `label` | `ReactNode` | ‚Äî | Primary label text. |
| `description` | `ReactNode` | ‚Äî | Optional secondary copy displayed below the label. |
| `disabled` | `boolean` | `false` | Disables this individual item. |
