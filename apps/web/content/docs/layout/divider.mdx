---
title: Divider
description: A patterned horizontal or vertical divider band for separating sections and panes. Use Separator for a plain one-pixel line.
---

## Import

```tsx
import { Divider } from "@sigil-ui/components";
```

## Usage

```tsx
<Divider pattern="vertical" size="md" />
<Divider pattern="diagonal" size="sm" />
<Divider orientation="vertical" pattern="diagonal" size="md" />
```

## Horizontal

```tsx
<Divider pattern="vertical" />
<Divider pattern="diagonal" size="sm" />
<Divider pattern="vertical" size="lg" />
```

## Vertical

```tsx
<Stack direction="row" gap={16} className="h-24">
  <span>Left</span>
  <Divider orientation="vertical" pattern="diagonal" size="sm" />
  <span>Right</span>
</Stack>
```

## With label

```tsx
<Divider pattern="vertical" label="01" showCross />
```

## Token integration

| CSS Variable | Used for |
|---|---|
| `--s-border-muted` | Pattern and edge border color |
| `--s-border` | Cross color |
| `--s-cross-stroke` | Cross stroke |
| `--s-background` | Label background |
| `--s-font-mono` | Label font |
| `--s-text-muted` | Label text |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Divider direction |
| `pattern` | `"vertical" \| "diagonal"` | `"vertical"` | Pattern variant |
| `size` | `"xs" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"` | Band thickness |
| `opacity` | `number` | `0.15` | Pattern opacity |
| `scale` | `number` | `1` | Pattern scale multiplier |
| `showBorders` | `boolean` | `true` | Show edge borders |
| `showCross` | `boolean` | `false` | Show endpoint cross marks |
| `label` | `ReactNode` | — | Center label |
| `fadeEdges` | `boolean` | `true` | Mask pattern near endpoints |
| `decorative` | `boolean` | `true` | Hide from accessibility tree |

## Accessibility

- Decorative by default.
- Set `decorative={false}` to render `role="separator"` with `aria-orientation`.
- Use `Separator` when you need a plain semantic line inside menus, forms, and small UI groups.
