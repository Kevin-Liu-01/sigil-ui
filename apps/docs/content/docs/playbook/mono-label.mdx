## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `MonoLabelVariant` | `muted` | Visual variant. |

| `as` | `"span" | "p" | "label" | "div" | "dt" | "dd"` | `span` | Render as a different element (e.g. "p", "label", "div"). |
