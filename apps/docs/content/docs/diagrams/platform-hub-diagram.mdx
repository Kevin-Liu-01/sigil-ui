## Import

```tsx
import { PlatformHubDiagram } from "@sigil-ui/components";
```

## Usage

```tsx
<PlatformHubDiagram
  left={[{ label: "Inputs", items: [{ label: "Tokens" }] }]}
  center={{ label: "Platform" }}
  right={[{ label: "Outputs", items: [{ label: "Components" }] }]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `left` | `PlatformGroup[]` | — | Left-side groups |
| `center` | `{ label: string; icon?: ReactNode; items?: PlatformItem[] }` | — | Central hub |
| `right` | `PlatformGroup[]` | — | Right-side groups |

## Token integration

| CSS Variable | Purpose |
