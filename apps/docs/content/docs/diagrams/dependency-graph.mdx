## Usage

```tsx
<DependencyGraph
  nodes={[
    { id: "app", label: "App", x: 250, y: 40 },
    { id: "api", label: "API", x: 120, y: 150 },
    { id: "db", label: "DB", x: 380, y: 150 },
    { id: "cache", label: "Cache", x: 250, y: 260, group: "infra" },
  ]}
  edges={[
    { from: "app", to: "api" },
    { from: "app", to: "db" },
    { from: "api", to: "cache", label: "reads", dashed: true },
    { from: "db", to: "cache", label: "syncs" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `nodes` | `DepNode[]` | — | Node positions and labels |
| `edges` | `DepEdge[]` | — | Directed edges between nodes |
| `width` | `number` | `500` | SVG width |
| `height` | `number` | `400` | SVG height |
| `nodeRadius` | `number` | `28` | Node circle radius |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Node stroke and edge lines |
| `--s-surface` | Node fill |
| `--s-text` | Node label text |
| `--s-text-muted` | Edge labels and group text |
| `--s-font-mono` | Label font family |
