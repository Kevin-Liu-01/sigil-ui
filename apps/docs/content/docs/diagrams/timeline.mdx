---
title: Timeline
description: "A vertical or horizontal timeline with dated entries and descriptions."
---

## Import

```tsx
import { Timeline } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <Timeline
    entries={[
      { date: "2026-04", title: "v2.0 Launch", description: "Complete design token overhaul and 200+ token-driven components." },
      { date: "2026-03", title: "Beta Release", description: "Public beta with early preset coverage." },
      { date: "2026-02", title: "Alpha", description: "Internal testing with core token system." },

## Usage

```tsx
<Timeline
  entries={[
    { date: "2026-04", title: "Launch", description: "Public release" },
    { date: "2026-03", title: "Beta", description: "Testing phase" },
  ]}
  variant="vertical"
/>
```

## Horizontal variant

```tsx
<Timeline variant="horizontal" entries={entries} />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `entries` | `TimelineEntry[]` | — | Timeline entries with date, title, and optional description |
| `variant` | `"vertical" \| "horizontal"` | `"vertical"` | Layout orientation |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Timeline dot color |
| `--s-border` | Vertical line and horizontal connector |
| `--s-text` | Entry title |
| `--s-text-muted` | Date and description text |
| `--s-font-mono` | Date font |
