---
title: SankeyDiagram
description: "A Sankey flow diagram showing weighted relationships between source and target nodes."
---

## Import

```tsx
import { SankeyDiagram } from "@sigil-ui/components";
```

## Usage

```tsx
<SankeyDiagram
  sources={[{ label: "Organic", value: 120 }]}
  targets={[{ label: "Signup", value: 100 }]}
  links={[{ source: 0, target: 0, value: 70 }]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `sources` | `SankeyNode[]` | — | Left-side source nodes |
| `targets` | `SankeyNode[]` | — | Right-side target nodes |
| `links` | `SankeyLink[]` | — | Weighted links between source and target indices |
| `width` | `number` | `400` | SVG width |
| `height` | `number` | `200` | SVG height |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default node and link color |
| `--s-text-muted` | Node labels |
| `--s-font-mono` | Label font family |
