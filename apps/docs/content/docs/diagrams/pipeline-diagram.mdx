---

## Import

```tsx
import { PipelineDiagram } from "@sigil-ui/components";
```

## Usage

```tsx
<PipelineDiagram
  steps={[
    { label: "Source", status: "complete" },
    { label: "Build", status: "active" },
    { label: "Deploy", status: "pending" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `steps` | `PipelineStep[]` | — | Pipeline stages with label and status |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Active step color |
| `--s-success` | Complete step color |
| `--s-border` | Connector and pending step border |
| `--s-surface` | Step background |
| `--s-text` | Step label |
| `--s-text-muted` | Pending step text |
| `--s-radius-md` | Step border radius |
