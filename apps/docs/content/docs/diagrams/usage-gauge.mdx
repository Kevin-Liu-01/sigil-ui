
## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `number` | — | Current value |
| `max` | `number` | `100` | Maximum value |
| `label` | `string` | — | Label below percentage |
| `unit` | `string` | — | Unit shown on scale |
| `width` | `number` | `160` | SVG width |
| `height` | `number` | `120` | SVG height |
| `color` | `string` | — | Override arc color |
| `autoColor` | `boolean` | `true` | Auto-color by threshold |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Normal usage color |
| `--s-warning` | Warning threshold (>70%) |
| `--s-error` | Critical threshold (>90%) |
| `--s-border` | Track background |
| `--s-text` | Percentage text |
| `--s-text-muted` | Label and scale text |
| `--s-font-mono` | Value font |
