## Import

```tsx
import { ComparisonTable } from "@sigil-ui/components";
```

## Usage

```tsx
<ComparisonTable
  columns={["Sigil", "shadcn/ui", "Chakra"]}
  features={[
    { name: "Design tokens", values: { Sigil: true, "shadcn/ui": false, Chakra: true } },
    { name: "Components", values: { Sigil: "65+", "shadcn/ui": "40+", Chakra: "50+" } },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `features` | `ComparisonFeature[]` | — | Feature rows with boolean or string values |
| `columns` | `string[]` | — | Column names (must match keys in feature values) |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Table borders |
| `--s-border-style` | Border style |
| `--s-surface-elevated` | Row hover background |
| `--s-success` | Checkmark color |
| `--s-text` | Feature names and string values |
| `--s-text-muted` | Feature header and dash indicators |
| `--s-radius-md` | Container radius |
| `--s-shadow-sm` | Container shadow |
| `--s-duration-fast` | Hover transition |
