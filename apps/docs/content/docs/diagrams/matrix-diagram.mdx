## Usage

```tsx
<MatrixDiagram
  xAxisLabel="Effort →"
  yAxisLabel="Impact →"
  xLabels={["Low Effort", "High Effort"]}
  yLabels={["High Impact", "Low Impact"]}
  cells={[
    [
      { id: "quick", label: "Quick Wins", description: "Do first" },
      { id: "major", label: "Major Projects", description: "Plan carefully" },
    ],
    [
      { id: "fill", label: "Fill-ins", description: "Do if time" },
      { id: "avoid", label: "Thankless Tasks", description: "Avoid" },
    ],
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `cells` | `MatrixCell[][]` | — | 2D array of cell content |
| `xLabels` | `string[]` | — | Column labels |
| `yLabels` | `string[]` | — | Row labels |
| `xAxisLabel` | `string` | — | X-axis title |
| `yAxisLabel` | `string` | — | Y-axis title |
