## Usage

```tsx
<Diagram showGrid gridVariant="dots" gridSize={24}>
  {/* Place DiagramNodes, DiagramConnectors, or custom content */}
</Diagram>
```

## Grid variants

- `"dots"` — Dot grid pattern
- `"lines"` — Line grid pattern
- `"cross"` — Combined dots and lines
- `"none"` — No grid

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `showGrid` | `boolean` | `false` | Show background grid |
| `gridVariant` | `"dots" \| "lines" \| "cross" \| "none"` | `"dots"` | Grid pattern style |
| `gridSize` | `number` | `24` | Grid cell size in px |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Container border |
| `--s-border-muted` | Grid pattern color |
| `--s-surface` | Container background |
| `--s-radius-card` | Container border radius |
