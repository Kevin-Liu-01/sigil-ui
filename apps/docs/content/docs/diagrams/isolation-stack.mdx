## Usage

```tsx
<IsolationStack
  layers={[
    { label: "Browser", width: "100%" },
    { label: "Sandbox", width: "82%" },
    { label: "Agent", width: "64%", accent: true },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `layers` | `IsolationLayer[]` | — | Stack layers |
| `columns` | `ComparisonColumn[]` | — | Optional comparison columns |
| `rows` | `ComparisonRow[]` | — | Optional comparison rows |
