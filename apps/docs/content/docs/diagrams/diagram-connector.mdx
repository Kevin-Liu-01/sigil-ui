## Import

```tsx
import { DiagramConnector } from "@sigil-ui/components";
```

## Usage

```tsx
<DiagramConnector direction="horizontal" variant="arrow" />
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `direction` | `"horizontal" \| "vertical"` | `"horizontal"` | Connector direction |
| `variant` | `"solid" \| "dashed" \| "arrow"` | `"solid"` | Line style |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Connector stroke |
| `--s-border-strong` | Arrow fill |
