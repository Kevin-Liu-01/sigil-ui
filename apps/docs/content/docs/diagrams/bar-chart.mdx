title: BarChart
description: "A bar chart with vertical or horizontal orientation and grouped bars."
---

## Import

```tsx
import { BarChart } from "@sigil-ui/components";
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `groups` | `BarChartGroup[]` | — | Grouped bar data |
| `direction` | `"vertical" \| "horizontal"` | `"vertical"` | Bar orientation |
| `showValues` | `boolean` | `true` | Show value labels |
| `showGrid` | `boolean` | `true` | Show gridlines |
| `maxBarSize` | `number` | `200` | Maximum bar size in px |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default first bar color |
| `--s-border` | Grid lines |
| `--s-text-muted` | Labels |
| `--s-font-mono` | Value and label font |
| `--s-radius-sm` | Bar corner radius |
| `--s-duration-normal` | Bar transition |
