## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | Metric label |
| `value` | `string \| number` | — | Main display value |
| `delta` | `string` | — | Change indicator (prefix with + or -) |
| `deltaLabel` | `string` | — | Context for the delta |
| `icon` | `ReactNode` | — | Optional icon |
| `sparkData` | `number[]` | — | Mini sparkline data |
| `sparkColor` | `string` | `var(--s-primary)` | Sparkline color |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default sparkline color |
| `--s-success` | Positive delta color |
| `--s-error` | Negative delta color |
| `--s-border` | Card border |
| `--s-surface` | Card background |
| `--s-text` | Value text |
| `--s-text-muted` | Label and delta label |
| `--s-font-mono` | Value and delta font |
| `--s-radius-card` | Card radius |
| `--s-shadow-sm` | Card shadow |
