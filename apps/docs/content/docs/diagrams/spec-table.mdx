## Usage

```tsx
<SpecTable
  title="GPU Specifications"
  groups={[
    {
      title: "Performance",
      rows: [
        { label: "TFLOPS", value: "312", unit: "FP16", highlight: true },
        { label: "Memory", value: "80", unit: "GB HBM3" },
        { label: "Bandwidth", value: "3.35", unit: "TB/s" },
      ],
    },
    {
      title: "Physical",
      rows: [
        { label: "TDP", value: "700", unit: "W" },
        { label: "Process", value: "4nm" },
      ],
    },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `rows` | `SpecRow[]` | — | Ungrouped specification rows |
| `groups` | `SpecGroup[]` | — | Grouped rows with section headers |
| `title` | `string` | — | Optional title header |
