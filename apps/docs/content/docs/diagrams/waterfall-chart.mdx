## Import

```tsx
import { WaterfallChart } from "@sigil-ui/components";
```

## Usage

```tsx
<WaterfallChart
  rows={[
    { label: "Before", steps: [{ label: "DNS", duration: 50 }, { label: "TLS", duration: 200 }] },
    { label: "After", accent: true, steps: [{ label: "DNS", duration: 10 }, { label: "TLS", duration: 20 }] },
  ]}
  unit="ms"
  badge="~10x faster"
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `rows` | `WaterfallRow[]` | — | Rows with label, steps array, and optional accent flag |
| `unit` | `string` | `"ms"` | Unit label for totals |
| `badge` | `string` | — | Badge text shown below (e.g. "~50x faster") |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Accent row value color |
| `--s-text` | Row label text |
| `--s-text-muted` | Non-accent row text |
| `--s-border` | Default step color |
| `--s-surface-elevated` | Bar track background |
| `--s-success` | Badge border and text |
| `--s-radius-sm` | Bar and segment radius |
| `--s-duration-normal` | Enter animation duration |
| `--s-ease-out` | Animation easing |
| `--s-ease-spring` | Badge spring easing |
| `--s-duration-fast` | Badge animation duration |
