---
title: ExplodedView
description: "An exploded view diagram showing separated layers of a system or component."
---

## Import

```tsx
import { ExplodedView } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <ExplodedView
    layers={[
      { label: "UI Layer", children: "React components with Tailwind" },
      { label: "Token Layer", children: "CSS custom properties" },
      { label: "Preset Layer", children: "259 configurable token values" },
    ]}
  />
</ComponentPreview>

## Usage

```tsx
<ExplodedView
  layers={[
    { label: "UI Layer", children: "React components" },
    { label: "Logic Layer", children: "Business rules" },
    { label: "Data Layer", children: "PostgreSQL" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `layers` | `ExplodedViewLayer[]` | — | Layers from top to bottom with label and content |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-border` | Layer borders |
| `--s-surface` | Layer background |
| `--s-text` | Layer label |
| `--s-text-muted` | Layer content text |
| `--s-radius-md` | Layer radius |
| `--s-shadow-sm` | Layer shadow |
