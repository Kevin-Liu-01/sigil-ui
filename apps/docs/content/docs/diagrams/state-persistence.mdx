## Usage

```tsx
<StatePersistence
  activeItem={{ label: "Theme", value: "saved", variant: "success" }}
  persistedState={[
    { label: "Preset", status: "restored" },
    { label: "Density", status: "restored" },
  ]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `activeItem` | `PersistenceActiveItem` | — | Primary active state item |
| `competitors` | `PersistenceActiveItem[]` | `[]` | Optional comparison items |
| `persistedState` | `PersistedStateRow[]` | `[]` | Persisted state rows |
| `activeLabel` | `string` | `"Status"` | Active section label |
| `stateLabel` | `string` | `"Preserved state"` | State section label |
