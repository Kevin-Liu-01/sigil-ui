
## Usage

```tsx
<ArchitectureDiagram
  title="Platform Architecture"
  layers={[
    { label: "Client", items: ["Web App", "Mobile App"] },
    { label: "Services", items: ["Auth", "Billing"] },
    { label: "Infrastructure", items: ["PostgreSQL", "Redis"] },
  ]}
  gap="1.5rem"
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `layers` | `ArchitectureLayer[]` | — | Layers from top (client) to bottom (infrastructure) |
| `title` | `string` | — | Diagram title |
| `gap` | `string` | `"1.5rem"` | Gap between layers |
