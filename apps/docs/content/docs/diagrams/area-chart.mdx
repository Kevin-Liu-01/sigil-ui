
## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `series` | `AreaChartSeries[]` | — | Data series |
| `width` | `number` | `500` | SVG width |
| `height` | `number` | `250` | SVG height |
| `stacked` | `boolean` | `false` | Stack areas |
| `showGrid` | `boolean` | `true` | Show grid lines |
| `curve` | `"smooth" \| "linear"` | `"smooth"` | Line interpolation |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default area color |
| `--s-border` | Grid lines |
| `--s-text-muted` | Axis labels |
| `--s-font-mono` | Label font |
