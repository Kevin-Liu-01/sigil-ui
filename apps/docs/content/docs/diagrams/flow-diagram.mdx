## Usage

```tsx
<FlowDiagram
  nodes={[
    { id: "input", label: "Input" },
    { id: "process", label: "Process", variant: "highlighted" },
    { id: "output", label: "Output", variant: "accent" },
  ]}
  connections={[
    { from: "input", to: "process", label: "validate" },
    { from: "process", to: "output" },
  ]}
  direction="horizontal"
  connector="dashed-animated"
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `nodes` | `FlowNode[]` | — | Process nodes with id, label, variant, and optional children |
| `connections` | `FlowConnection[]` | — | Connections between nodes with optional labels |
| `direction` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction |
| `connector` | `"solid" \| "dashed" \| "dashed-animated"` | `"dashed-animated"` | Connector line style |
