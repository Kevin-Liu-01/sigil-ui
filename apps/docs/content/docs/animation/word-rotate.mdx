## Import

```tsx
import { WordRotate } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <WordRotate words={["Design", "Build", "Ship"]} />
</ComponentPreview>

## Usage

```tsx
<WordRotate>WordRotate content</WordRotate>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `words` | `string[]` | — | Array of words/phrases to cycle through. |
| `duration` | `number` | `2500` | Time each word is shown in ms. |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-duration-fast` | Duration fast |
| `--s-ease-out` | Ease out |
| `--s-ease-default` | Ease default |
