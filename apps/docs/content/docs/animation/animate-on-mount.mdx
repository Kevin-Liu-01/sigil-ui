---
title: AnimateOnMount
description: "Wrapper that plays a CSS transition on mount (no scroll trigger). Lighter alternative to the GSAP-based AnimateOnScroll for simple mount reveals."
---

## Import

```tsx
import { AnimateOnMount } from "@sigil-ui/components";
```

## Usage

```tsx
<AnimateOnMount>...</AnimateOnMount>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `preset` | `PresetName` | `fadeUp` | Animation preset. |
| `delay` | `number` | `0` | Delay in ms. |
| `duration` | `number` | `400` | Duration in ms. |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-ease-default` | Ease default |
