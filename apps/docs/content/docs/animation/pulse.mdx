## Import

```tsx
import { Pulse } from "@sigil-ui/components";
```

## Usage

```tsx
<Pulse>...</Pulse>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `duration` | `number` | `2` | Pulse animation duration in seconds. |
| `scale` | `number` | `1.04` | Scale range of the pulse. |
