
## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `direction` | `"left" | "right"` | `right` | Diagonal cut direction. |
| `height` | `number` | `80` | Height of the diagonal section. |
| `fill` | `string` | — | Fill color (CSS value). Defaults to the surface token. |
