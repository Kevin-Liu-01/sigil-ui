## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `direction` | `"up" | "down" | "left" | "right"` | `up` | Direction the triangle points. |
| `size` | `"sm" | "md" | "lg" | "xl"` | `md` | Triangle size. |
