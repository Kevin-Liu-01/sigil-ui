## Usage

```tsx
<Shape>Shape content</Shape>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `ShapeVariant` | `diamond` | Shape type. |
| `size` | `ShapeSize` | `md` | Size. |
| `filled` | `boolean` | — | Fill the shape with color. |
| `strokeWidth` | `number` | `1.5` | Custom stroke width. |
