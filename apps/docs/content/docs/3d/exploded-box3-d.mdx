## Import

```tsx
import { ExplodedBox3D } from "@sigil-ui/components";
```

## Usage

```tsx
<ExplodedBox3D>...</ExplodedBox3D>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `size` | `number` | `80` | Size of the front/back square in px. |
| `depth` | `number` | `22` | Depth of the side faces in px. |
| `gap` | `number` | `10` | Gap between the separated faces in px. |
| `variant` | `"container" | "card" | "panel"` | `container` | Preset variant. |
| `showLabels` | `boolean` | `false` | Show small face labels for diagram use. |
