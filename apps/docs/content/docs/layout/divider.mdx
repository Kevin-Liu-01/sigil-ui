## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `orientation` | `"horizontal" | "vertical"` | `horizontal` | Divider orientation. |
| `decorative` | `boolean` | `true` | If true, this is purely decorative (aria-hidden). |
