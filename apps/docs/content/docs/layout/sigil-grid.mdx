---
title: SigilGrid
description: "Grid with visible cross marks at cell intersections. Uses a pseudo-element background to render the sigil pattern."
---

## Import

```tsx
import { SigilGrid, SigilGridCell } from "@sigil-ui/components";
```

## Usage

```tsx
<SigilGrid>...</SigilGrid>
```

## Sub-components

This component includes the following parts:

- `<SigilGridCell>`

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `columns` | `2 | 3 | 4 | 5 | 6` | `3` | Number of columns. |
| `gap` | `string | number` | — | Gap between cells. |
