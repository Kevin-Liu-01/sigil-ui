title: PageGrid
description: "A PageGrid component from the Sigil design system."

## Import

```tsx
import { PageGrid } from "@sigil-ui/components";
```

## Usage

```tsx
<PageGrid>...</PageGrid>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `showRails` | `boolean` | — | Show vertical rail marks on left/right edges. |
| `showCross` | `boolean` | — | Show cross marks at grid intersections. |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-rail-gap` | Rail gap |
| `--s-border-muted` | Border muted |
| `--s-grid-cell` | Grid cell |
