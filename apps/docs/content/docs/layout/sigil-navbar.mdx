## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `"full" | "inline"` | `full` | "full" — renders its own 5-column grid so the nav content aligns with the SigilPageGrid content column, with structural gutter borders flanking the nav. "inline" — a plain `<nav>` with a bottom border, suitable for nesting inside a SigilPageGrid content column. / |

| `railGap` | `number` | — | — |

| `showMarginLines` | `boolean` | — | — |
