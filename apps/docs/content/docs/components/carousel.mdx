## Import

```tsx
import { Carousel, CarouselContent, CarouselItem, CarouselPrevious, CarouselNext } from "@sigil-ui/components";
```

## Usage

```tsx
<Carousel>...</Carousel>
```

## Sub-components

This component includes the following parts:

- `<CarouselContent>`
- `<CarouselItem>`
- `<CarouselPrevious>`
- `<CarouselNext>`
