```tsx
import { FileDropzone } from "@sigil-ui/components";
```

```tsx
<FileDropzone label="Upload files" description="Drag files here or browse." />
```

## CLI

```bash
npx @sigil-ui/cli add file-dropzone
```

## Token Integration

FileDropzone follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
