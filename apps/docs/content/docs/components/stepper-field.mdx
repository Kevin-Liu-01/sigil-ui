## Import

```tsx
import { StepperField } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <StepperField defaultValue={3} />
</ComponentPreview>

## Usage

```tsx
<StepperField defaultValue={3} />
```

## CLI

```bash
npx @sigil-ui/cli add stepper-field
```

## Token Integration

StepperField follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
