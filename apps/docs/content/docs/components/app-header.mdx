## Import

```tsx
import { AppHeader } from "@sigil-ui/components";
```

```tsx
<AppHeader title="Components" description="Browse the catalog." actions={<Button>New</Button>} />
```

## CLI

```bash
npx @sigil-ui/cli add app-header
```

## Token Integration

AppHeader follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
