## Import

```tsx
import { SkeletonCard } from "@sigil-ui/components";
```

## Preview

## Usage

```tsx
<SkeletonCard />
```

## CLI

```bash
npx @sigil-ui/cli add skeleton-card
```

## Token Integration

SkeletonCard follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
