## Import

```tsx
import { MultiSelect } from "@sigil-ui/components";
```

## Usage

```tsx
<MultiSelect options={[{ value: "design", label: "Design" }, { value: "docs", label: "Docs" }, { value: "data", label: "Data" }]} defaultValue={["design", "data"]} />
```

## CLI

```bash
npx @sigil-ui/cli add multi-select
```

## Token Integration

MultiSelect follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
