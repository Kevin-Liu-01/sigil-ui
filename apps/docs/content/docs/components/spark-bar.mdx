## Import

```tsx
import { SparkBar } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <SparkBar values={[4, 8, 6, 12, 10, 16]} />

## Usage

```tsx
<SparkBar values={[4, 8, 6, 12, 10, 16]} />
```

## CLI

```bash
npx @sigil-ui/cli add spark-bar
```

## Token Integration

SparkBar follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
