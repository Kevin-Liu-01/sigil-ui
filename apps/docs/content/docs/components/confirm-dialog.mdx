## Import

```tsx
import { ConfirmDialog } from "@sigil-ui/components";
```

## Usage

```tsx
<ConfirmDialog trigger={<Button variant="outline">Confirm action</Button>} title="Confirm action" description="Use for destructive or important decisions." />
```

## CLI

```bash
npx @sigil-ui/cli add confirm-dialog
```

## Token Integration

ConfirmDialog follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
