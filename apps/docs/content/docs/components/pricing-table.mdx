## Import

```tsx
import { PricingTable } from "@sigil-ui/components";
```

## Usage

```tsx
<PricingTable
  columns={[
    { name: "Free", price: "$0" },
    { name: "Pro", price: "$19", period: "mo", highlighted: true, badge: "Popular" },
  ]}
  features={features}
  categories={[{ label: "Usage", startIndex: 0 }]}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `columns` | `PricingColumn[]` | — | Tier definitions with name, price, badge |
| `features` | `PricingFeature[]` | — | Feature rows with boolean or string values per tier |
| `categories` | `object[]` | — | Optional category grouping headers |
