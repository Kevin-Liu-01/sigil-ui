
```tsx
<ContextPanel trigger={<Button variant="outline">Open panel</Button>}>Contextual content</ContextPanel>
```

## CLI

```bash
npx @sigil-ui/cli add context-panel
```

## Token Integration

ContextPanel follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
