title: Calendar

## Import

```tsx
import { Calendar } from "@sigil-ui/components";
```

## Usage

<Calendar mode="single" />

### With dropdown navigation (default)

The calendar defaults to `captionLayout="dropdown"` with a 100-year range, letting users jump to any month or year quickly.

<Calendar mode="single" />

### Label-only caption

<Calendar mode="single" captionLayout="label" />

### Fixed height

The calendar uses `fixedWeeks` by default — it always renders 6 rows so height never shifts when navigating between months.
