## Import

```tsx
import { ActivityFeed } from "@sigil-ui/components";
```

## Usage

```tsx
<ActivityFeed
  entries={[
    { id: "1", title: "Deploy succeeded", timestamp: "2m ago", variant: "success" },
    { id: "2", title: "Build failed", timestamp: "1h ago", variant: "error" },
  ]}
  maxEntries={10}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `entries` | `ActivityEntry[]` | — | Feed entries with title, description, timestamp, variant |
| `maxEntries` | `number` | — | Truncate after N entries (shows "+X more") |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-success` | Success variant dot |
| `--s-warning` | Warning variant dot |
| `--s-error` | Error variant dot |
| `--s-info` | Info variant dot |
| `--s-border` | Timeline vertical line |
| `--s-border-strong` | Default variant dot |
| `--s-text` | Entry title |
| `--s-text-muted` | Description, timestamp, icons |
| `--s-font-mono` | Timestamp font family |
