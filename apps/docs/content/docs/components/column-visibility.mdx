## Import

```tsx
import { ColumnVisibility } from "@sigil-ui/components";
```

## Usage

```tsx
<ColumnVisibility columns={[{ id: "name", label: "Name" }, { id: "status", label: "Status", visible: false }]} />
```

## CLI

```bash
npx @sigil-ui/cli add column-visibility
```

## Token Integration

ColumnVisibility follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
