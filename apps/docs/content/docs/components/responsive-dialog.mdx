## Import

```tsx
import { ResponsiveDialog } from "@sigil-ui/components";
```

## Usage

```tsx
<ResponsiveDialog trigger={<Button>Open responsive dialog</Button>} title="Responsive dialog" description="A dialog preset for app workflows." />
```

## CLI

```bash
npx @sigil-ui/cli add responsive-dialog
```

## Token Integration

ResponsiveDialog follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
