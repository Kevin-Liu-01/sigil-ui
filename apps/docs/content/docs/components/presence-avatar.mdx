```tsx
import { PresenceAvatar } from "@sigil-ui/components";
```

```tsx

```

## CLI

```bash
npx @sigil-ui/cli add presence-avatar
```

## Token Integration

PresenceAvatar follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
