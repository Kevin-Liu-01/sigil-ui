---
title: Center
description: Centers its children both horizontally and vertically using flexbox.
---

## Import

```tsx
import { Center } from "@sigil-ui/components";
```

## Usage

```tsx
<Center className="h-40 border border-[var(--s-border)] border-dashed">
  <span>Centered content</span>
</Center>
```

## Inline variant

```tsx
<p>
  Text with an <Center inline>inline centered</Center> element.
</p>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `inline` | `boolean` | `false` | Use `inline-flex` instead of `flex` |
| `className` | `string` | — | Additional CSS classes |
