```tsx
import { StatCard } from "@sigil-ui/components";
```

```tsx
<StatCard label="Components" value="300+" change={<Trend value={50} />} />
```

## CLI

```bash
npx @sigil-ui/cli add stat-card
```

## Token Integration

StatCard follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
