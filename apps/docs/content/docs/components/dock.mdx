## Import

```tsx
import { Dock } from "@sigil-ui/components";
```

## Usage

```tsx
<Dock><Button size="icon" variant="ghost">A</Button><Button size="icon" variant="ghost">B</Button></Dock>
```

## CLI

```bash
npx @sigil-ui/cli add dock
```

## Token Integration

Dock follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
