title: MatrixDiagram

---

## Import

```tsx
import { MatrixDiagram } from "@sigil-ui/components";
```

## Usage

```tsx
<MatrixDiagram
  xAxisLabel="Effort"
  yAxisLabel="Impact"
  xLabels={["Low", "High"]}
  yLabels={["High", "Low"]}
  cells={matrix}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `cells` | `MatrixCell[][]` | — | 2D array of cell content |
| `xLabels` | `string[]` | — | Column labels |
| `yLabels` | `string[]` | — | Row labels |
| `xAxisLabel` | `string` | — | X-axis title |
| `yAxisLabel` | `string` | — | Y-axis title |
