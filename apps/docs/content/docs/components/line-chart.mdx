## Usage

```tsx
<LineChart
  series={[{ label: "Users", points: [{ label: "Mon", value: 120 }, ...] }]}
  showDots
  showGrid
  filled={false}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `series` | `LineChartSeries[]` | — | One or more data series to plot |
| `width` | `number` | `500` | SVG width |
| `height` | `number` | `250` | SVG height |
| `showDots` | `boolean` | `true` | Show circular nodes at each data point |
| `showGrid` | `boolean` | `true` | Show horizontal grid lines |
| `showXLabels` | `boolean` | `true` | Show x-axis labels |
| `showYLabels` | `boolean` | `true` | Show y-axis value labels |
| `filled` | `boolean` | `false` | Fill the area under the line |

## Token integration

| CSS Variable | Purpose |
|---|---|
| `--s-primary` | Default line and dot color |
| `--s-border` | Grid lines |
| `--s-surface` | Dot fill (white center) |
| `--s-text-muted` | Axis labels |
| `--s-font-mono` | Label font family |
