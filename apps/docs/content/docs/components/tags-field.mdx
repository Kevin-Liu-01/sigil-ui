```tsx
import { TagsField } from "@sigil-ui/components";
```

```tsx
<TagsField defaultValue={["base-ui", "tokens"]} />
```

## CLI

```bash
npx @sigil-ui/cli add tags-field
```

## Token Integration

TagsField follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
