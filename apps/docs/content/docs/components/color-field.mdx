## Import

```tsx
import { ColorField } from "@sigil-ui/components";
```

## Preview

<ComponentPreview>
  <ColorField defaultValue="#9b99e8" />

## Usage

```tsx
<ColorField defaultValue="#9b99e8" />
```

## CLI

```bash
npx @sigil-ui/cli add color-field
```

## Token Integration

ColorField follows the Sigil component rules: styles resolve from `var(--s-*)`, class names are mergeable through `className`, and the component is exported from `@sigil-ui/components`.
